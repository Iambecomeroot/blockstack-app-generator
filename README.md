## Installation

First, install [Yeoman](http://yeoman.io) and the blockstack app generator using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo generator-blockstack
```

Then create a new directory and generate your new project:

```bash
mkdir hello-blockstack && cd hello-blockstack
yo blockstack
```

Last, start the development server:

```bash
npm run start
```
